# 注释

dart语言中注释跟大多数语言一样使用反斜杠//

//这是主入口函数注释
main(){
    //这是函数体
}

除了反斜杠，像/* */和/** */可以用来注释大段注释文本。

/**
 * 这是主入口函数注释
 */
main(){
    /*
     * 这是函数体
     */
}
