# 循环语句 for while do-while break continue

## for

同JavaScript语言一样：

``` dart
  for (var i = 0; i < 100; i++) {
    print(i);
  }
```

执行步骤：

1. 先声明`var i = 0;`
2. 再判断`i < 100;`
3. 执行`print(i);`
4. 递增`i++`
5. 回到第二步判断，满足循环执行2、3、4，直到第二步判断为`false`.

## while do-while

while:
``` dart
  var i = 0;
  while (i < 10) {
    print(i);
    i++;
  }
```
do-while:
``` dart
  var i = 0;
  do {
    print(i);
    i++;
  } while (i < 10);
```

## break

跳出循环。

## continue

跳过本次循环，继续执行下次循环。
